## Legend

**Status:**
- `NONE` — no graphic exists
- `PLACEHOLDER` — graphic exists but is a flat colored shape, not matching style guide
- `PARTIAL` — some visual exists but incomplete (e.g., color dimming without fractures)

**Priority** (from the three lenses):
- `Readability` — Critical / Important / Low (Clarity agent)
- `Juice` — High / Medium / Low (Juice agent)
- `Cohesion Risk` — High / Medium / Low (Cohesion agent)

**Style Guide:**
- `COVERED` — described in `docs/design/graphics/*.md`
- `NEEDS DESCRIPTION` — no visual direction exists yet
