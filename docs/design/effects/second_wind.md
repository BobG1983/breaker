# SecondWind

Spawns an invisible bottom wall that bounces the bolt once, preventing bolt loss.

## Parameters

None.

## Behavior

Spawns an invisible wall entity at the bottom of the playfield. Permanent until used. When a bolt hits it, the bolt bounces back up, the wall despawns (single use), and a graphical flourish plays to show it was consumed.

## Reversal

Despawns the invisible wall entity if still alive.
