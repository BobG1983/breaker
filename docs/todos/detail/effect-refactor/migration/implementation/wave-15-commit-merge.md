# Wave 15: Commit, Merge, Push

## Goal
Land the effect refactor on develop.

## Steps

1. `git add` all changed files
2. Commit with message: `feat: implement effect system refactor and unified death pipeline`
3. `git flow update` — merge develop into feature branch
4. Resolve any merge conflicts
5. Re-run Basic Verification Tier after merge
6. `git flow feature finish` — merge to develop, delete branch
7. `git push`
