# Name
BumpTarget

# Syntax
```rust
enum BumpTarget {
    Bolt,
    Breaker,
}
```

# Description
- Bolt: The bolt that was bumped. See [bump-target](../../ron-syntax/participants/bump-target.md)
- Breaker: The breaker that did the bumping. See [bump-target](../../ron-syntax/participants/bump-target.md)
