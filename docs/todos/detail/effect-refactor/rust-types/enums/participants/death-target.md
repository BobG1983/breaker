# Name
DeathTarget

# Syntax
```rust
enum DeathTarget {
    Victim,
    Killer,
}
```

# Description
- Victim: The entity that died. See [death-target](../../ron-syntax/participants/death-target.md)
- Killer: The entity that caused the death. May not exist for environmental deaths. See [death-target](../../ron-syntax/participants/death-target.md)
