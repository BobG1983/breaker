# Name
BoltLostTarget

# Syntax
```rust
enum BoltLostTarget {
    Bolt,
    Breaker,
}
```

# Description
- Bolt: The bolt that was lost. See [bolt-lost-target](../../ron-syntax/participants/bolt-lost-target.md)
- Breaker: The breaker that lost the bolt. See [bolt-lost-target](../../ron-syntax/participants/bolt-lost-target.md)
