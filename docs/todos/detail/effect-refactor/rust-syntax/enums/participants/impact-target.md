# Name
ImpactTarget

# Syntax
```rust
enum ImpactTarget {
    Impactor,
    Impactee,
}
```

# Description
- Impactor: The entity that initiated the collision (e.g. the bolt that hit a cell). See [impact-target](../../ron-syntax/participants/impact-target.md)
- Impactee: The entity that was hit (e.g. the cell that was hit by a bolt). See [impact-target](../../ron-syntax/participants/impact-target.md)
