# ImpactTarget

Roles in a collision event. Used by: Impacted, ImpactOccurred.

| Name | RON Syntax | Description |
|------|-----------|-------------|
| `Impactor` | `ImpactTarget::Impactor` | The entity that initiated the collision (e.g. the bolt that hit a cell). |
| `Impactee` | `ImpactTarget::Impactee` | The entity that was hit (e.g. the cell that was hit by a bolt). |
