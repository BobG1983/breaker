# Participant Types

| Participant Type | Description |
|-----------------|-------------|
| `BumpTarget` | Roles in a bump event (bolt hitting breaker). See [bump-target.md](bump-target.md). |
| `ImpactTarget` | Roles in a collision event (two entities colliding). See [impact-target.md](impact-target.md). |
| `DeathTarget` | Roles in a death event (an entity dying). See [death-target.md](death-target.md). |
| `BoltLostTarget` | Roles in a bolt lost event (bolt falling off bottom). See [bolt-lost-target.md](bolt-lost-target.md). |
