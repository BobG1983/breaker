# DeathTarget

Roles in a death event. Used by: Died, Killed, DeathOccurred.

| Name | RON Syntax | Description |
|------|-----------|-------------|
| `Victim` | `DeathTarget::Victim` | The entity that died. |
| `Killer` | `DeathTarget::Killer` | The entity that caused the death. May not exist (environmental deaths have no killer). |
