# BumpTarget

Roles in a bump event. Used by: PerfectBumped, EarlyBumped, LateBumped, Bumped, PerfectBumpOccurred, EarlyBumpOccurred, LateBumpOccurred, BumpOccurred, BumpWhiffOccurred, NoBumpOccurred.

| Name | RON Syntax | Description |
|------|-----------|-------------|
| `Bolt` | `BumpTarget::Bolt` | The bolt that was bumped. |
| `Breaker` | `BumpTarget::Breaker` | The breaker that did the bumping. |
