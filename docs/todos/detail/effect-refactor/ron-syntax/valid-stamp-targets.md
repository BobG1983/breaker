# Valid StampTargets

First argument to `Stamp()`. Which entity or entities receive the Tree.

| StampTarget | Meaning |
|-------------|---------|
| `Bolt` | Primary bolt entity |
| `Breaker` | Primary breaker entity |
| `ActiveBolts` | All bolt entities that exist right now |
| `EveryBolt` | All existing bolts + all bolts spawned in the future |
| `PrimaryBolts` | All bolts with the PrimaryBolt marker |
| `ExtraBolts` | All bolts with the ExtraBolt marker |
| `ActiveCells` | All cell entities that exist right now |
| `EveryCell` | All existing cells + all cells spawned in the future |
| `ActiveWalls` | All wall entities that exist right now |
| `EveryWall` | All existing walls + all walls spawned in the future |
| `ActiveBreakers` | All breaker entities that exist right now |
| `EveryBreaker` | All existing breakers + all breakers spawned in the future |
