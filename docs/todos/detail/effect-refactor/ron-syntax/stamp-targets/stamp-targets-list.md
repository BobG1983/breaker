# Valid StampTargets

First argument to `Stamp()`. Which entity or entities receive the Tree.

| StampTarget | Meaning |
|-------------|---------|
| [`Bolt`](bolt.md) | Primary bolt entity |
| [`Breaker`](breaker.md) | Primary breaker entity |
| [`ActiveBolts`](active-bolts.md) | All bolt entities that exist right now |
| [`EveryBolt`](every-bolt.md) | All existing bolts + all bolts spawned in the future |
| [`PrimaryBolts`](primary-bolts.md) | All bolts with the PrimaryBolt marker |
| [`ExtraBolts`](extra-bolts.md) | All bolts with the ExtraBolt marker |
| [`ActiveCells`](active-cells.md) | All cell entities that exist right now |
| [`EveryCell`](every-cell.md) | All existing cells + all cells spawned in the future |
| [`ActiveWalls`](active-walls.md) | All wall entities that exist right now |
| [`EveryWall`](every-wall.md) | All existing walls + all walls spawned in the future |
| [`ActiveBreakers`](active-breakers.md) | All breaker entities that exist right now |
| [`EveryBreaker`](every-breaker.md) | All existing breakers + all breakers spawned in the future |
