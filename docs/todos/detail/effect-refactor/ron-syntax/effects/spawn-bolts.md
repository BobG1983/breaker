# Name
SpawnBolts

# Parameters
`SpawnBoltsConfig`

# Description
Spawns additional bolts from the source entity's position, each flying in a random direction. Spawned bolts are tagged as extra bolts (not primary). They can optionally have a lifespan -- after which they despawn automatically -- and can optionally inherit the first primary bolt's effect trees, so they benefit from the same chips. See [SpawnBoltsConfig](../configs/spawn-bolts-config.md).
