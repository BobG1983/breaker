# Name
ChainBolt

# Parameters
`ChainBoltConfig`

# Description
Spawns a new bolt tethered to the source bolt by a distance constraint. The two bolts can move independently but are pulled back toward each other if they exceed the tether distance. This creates a paired-bolt dynamic where the bolts sweep an area between them. See [ChainBoltConfig](../configs/chain-bolt-config.md).
