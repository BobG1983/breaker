# Name
Pulse

# Parameters
`PulseConfig`

# Description
Attaches a periodic shockwave emitter to the entity. At a regular interval, a shockwave ring expands outward from the entity's position, damaging all cells it passes through. This turns the bolt (or breaker) into a mobile area-damage platform -- just being near cells damages them over time. See [PulseConfig](../configs/pulse-config.md).
