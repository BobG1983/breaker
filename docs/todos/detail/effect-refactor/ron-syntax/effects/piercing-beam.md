# Name
PiercingBeam

# Parameters
`PiercingBeamConfig`

# Description
Fires an instant damage beam along the bolt's current velocity direction. All cells within a rectangle extending from the bolt's position in its travel direction, with a configured width, take damage simultaneously. This is a line-of-fire attack -- the bolt doesn't change direction, it just damages everything in its path at the moment the effect fires. See [PiercingBeamConfig](../configs/piercing-beam-config.md).
