# Name
SecondWind

# Parameters
`SecondWindConfig` — See [SecondWindConfig](../configs/second-wind-config.md)

# Description
Spawns an invisible one-shot wall at the bottom of the play area. The first bolt that would fall off the bottom bounces off this wall instead, saving it. The wall is consumed after one bounce and disappears. Unlike Shield, the player can't see it -- it's a safety net.
