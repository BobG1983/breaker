# Name
Shield

# Parameters
`ShieldConfig`

# Description
Spawns a visible wall along the bottom of the play area that reflects bolts, preventing bolt loss for its duration. Each time a bolt bounces off the shield, the shield's remaining time is reduced by a reflection cost. If a shield already exists when this effect fires, its timer is reset instead of spawning a second one. See [ShieldConfig](../configs/shield-config.md).
