# Name
Explode

# Parameters
`ExplodeConfig`

# Description
Instant area damage burst at the entity's position. All cells within range take a flat damage amount immediately. Unlike Shockwave, there is no expanding ring -- the damage is instant. And unlike most damage effects, the damage is flat -- it is NOT modified by damage boosts. This makes Explode predictable regardless of other active effects. See [ExplodeConfig](../configs/explode-config.md).
