# Name
Anchor

# Parameters
`AnchorConfig`

# Description
Enables the plant mechanic on the breaker. When the breaker stands still for a configured delay, it "plants" -- gaining a boosted bump force multiplier +1 piercing, and a wider perfect timing window. This rewards patient positioning: stop moving, wait for the bolt, and get a more powerful and more forgiving bump. The plant state ends when the breaker moves again. See [AnchorConfig](../configs/anchor-config.md).
