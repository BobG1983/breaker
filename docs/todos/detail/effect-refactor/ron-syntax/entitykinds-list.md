# EntityKind

Entity type filter used by Trigger parameters and Spawned.

| EntityKind | RON Syntax | Description |
|------------|-----------|-------------|
| `Cell` | `Cell` | Matches cell entities |
| `Bolt` | `Bolt` | Matches bolt entities |
| `Wall` | `Wall` | Matches wall entities |
| `Breaker` | `Breaker` | Matches breaker entities |
| `Any` | `Any` | Matches any entity type |
